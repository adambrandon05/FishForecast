# 🎣 FishForecast Bot

A Discord bot that sends personalized fishing information to users based on their preferences.

---

## 🚀 Project Overview

This project is focused on building a system that can:

- Send daily messages to users on Discord
- Personalize those messages based on user input
- Eventually incorporate weather data and simple predictive logic

The goal is to build this in stages while learning core concepts like APIs, scheduling, databases, and hosting.
